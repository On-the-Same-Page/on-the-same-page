About:

We love reading books. In today’s age, many have difficulties finding the right one to get lost in - having too many options can make it hard to make a satisfactory choice. On the Same Page visualizes hundreds of books using a filterable scatterplot and provides detailed information to assist you in your search for the next read! Have fun exploring On the Same Page and maybe you will find your next favorite somewhere in the plot?

Data:

The data was obtained from Goodreads website. Goodreads has a database with dozens of millions of books, so we decided to use a list of "the best books ever", according to the general Goodreads community. We then scraped the top one thousand books from the list, using this scraper.

We needed some information that was beyond the scope of the scraper, so we coded additional scraping scripts to get those (such as book covers, publication year, a better list of authors etc.). Data was then cleaned, wrangled and shaped to best fit our visualization needs. All that was done with R, as well as exploratory data analysis and visualizations prototyping. All code related to these activities are available at our data-prep repository. You can also explore the resulting JSON file used in this application. The data was last updated on March 12, 2023.

Methods:

On the Same Page is built using Angular and D3.js. For further information, check our repository on GitHub.

Learning Objectives Reached

After this class, we are able to:

  • Design: Collaborate as groups to create fully-functioning information visualization systems that facilitate actionable insight through interactive data transformations, visual mappings, and view transformations;
  • Defend: Defend your design choices through deep domain knowledge;
  • Critique: Constructively criticize other information visualization systems using domain theory and practices;
  • Demonstrate: Explain and demonstrate your information visualization systems to wide audiences, from novices to experts;
  • Evaluate: Elicit constructive criticism from users of your information visualization systems.

Besides that and our improved general skills in information visualization, some important lessons we learned from this project:

  • Data cleaning, shaping and exploration are essential activities throughout a visualization project, so familiarity with a data analysis environment is key (R/ggplot2, Python/Pandas/Altair/matplotlib, Observable/Vega Lite/Plot etc.);
  • The importance of teamwork and open, respectful communication;
  • The visualization mantra: overview first, filter and zoom, details on demand;
  • The importance (and challenges) of user evaluation.
