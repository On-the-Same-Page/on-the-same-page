Have fun exploring On the Same Page and maybe you will find your next favorite somewhere in the plot?

Data:

The data was obtained from Goodreads website. Goodreads has a database with dozens of millions of books, so we decided to use a list of "the best books ever", according to the general Goodreads community. We then scraped the top one thousand books from the list, using this scraper.

We needed some information that was beyond the scope of the scraper, so we coded additional scraping scripts to get those (such as book covers, publication year, a better list of authors etc.). Data was then cleaned, wrangled and shaped to best fit our visualization needs. All that was done with R, as well as exploratory data analysis and visualizations prototyping. All code related to these activities are available at our data-prep repository. The data was last updated on March 12, 2023.

Methods:

On the Same Page is built using Angular and D3.js. For further information, check our repository on GitHub.
