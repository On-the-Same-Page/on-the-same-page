Have fun exploring On the Same Page and maybe you will find your next favorite somewhere in the plot?
Data:
The data was scraped from the website goodreads.com (https://www.goodreads.com/list/show/1.Best_Books_Ever) using this scraper: https://github.com/havanagrawal/GoodreadsScraper. It contains 975 books. We did some data cleaning to come up with the final dataset. The data was last updated on March 21, 2023.
Methods:
On the Same Page is built using Angular and D3.js. For further information, check our repository on GitHub.
